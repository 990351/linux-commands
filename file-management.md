### File Management Commands ###


## cp
Copies files or directories

Examples:
cp file1.txt file2.txt
cp -r dir1 dir2


## mv
Moves or renames files

Examples:
mv old.txt new.txt
mv file.txt /tmp/


## stat
Shows detailed file information

Example:
stat file.txt


## head
Shows first lines of a file

Example:
head file.txt


## tail
Shows last lines of a file

Example:
tail file.txt
tail -f logfile.log
