# Basic Linux Commands

1. pwd
   Shows the present working directory
 
   Example:
      pwd
   Output:
       /home/user

2. ls
   Lists files and directories

   Examples:
       ls
       ls -l
       ls -a

 3. cd
    Changes directory

Examples:
    cd Documents
    cd ..
    cd /

 3. mkdir
    Creates a directory

    Examples:
        mkdir test

 4. touch
    Creates an empty file.

    Example:
        touch file.txt
    
 5. cat
     versatile utility used primarily to display the contents of files, combine multiple files, and create new files directly from the command line

    Example:
         cat file.txt
         cat file1.txt file2.txt
         cat file1.txt file2.txt > combined.txt
         cat content.txt >> existingfile.txt

 6.  rm
     Deletes files or directories

     Examples:
          rm file.txt
          rm -r folder_name
